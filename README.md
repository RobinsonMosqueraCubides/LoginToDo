# Proyecto LoginToDo

## Descripción
Este proyecto consiste en una aplicación web sencilla que permite a los usuarios iniciar sesión y gestionar una lista de tareas.

## 📌 Requisitos de Acceso
Para iniciar sesión en la aplicación, utiliza las siguientes credenciales:
- **Usuario:** `user1`
- **Contraseña:** `user1`

Una vez iniciada la sesión, el usuario será redirigido a la lista de tareas, donde podrá agregar, marcar como completadas o eliminar tareas.
